Projekt przykładowej aplikacji rejestracji wizyt.

Aby uruchomić aplikację:
1. Zbuduj rozwiązanie: `dotnet build ClinicApp.sln`.
2. Uruchom API: `dotnet run --project Clinic.WebApi`.
3. Uruchom aplikację Blazor Server: `dotnet run --project Clinic.BlazorServer`.
4. Uruchom aplikację Blazor WebAssembly: `dotnet run --project Clinic.BlazorWasm`.

Domyślnie baza danych SQLite tworzona jest w pliku `clinic.db`.
